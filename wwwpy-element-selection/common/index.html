Pseudo-Element Border Highlight

This technique uses an absolutely-positioned pseudo-element (::after) to draw a custom highlight around the target element without affecting layout.

.highlight-frame { position: relative; } .highlight-frame::after { content: ""; position: absolute; top: -4px; left: -4px; right: -4px; bottom: -4px; border: 2px dashed #ff7f50; border-radius: inherit; pointer-events: none; }
