Pushable Sidebar Demo

This demo shows how to use the PushableSidebar library to create sidebars that push content away instead of overlapping it.

Control Panel


How to Use

The PushableSidebar library creates a custom element that you can use in your HTML or create programmatically.

Method 1: HTML Declaration

<pushable-sidebar position="left" width="300px" theme="light">
  <div>Your sidebar content here</div>
</pushable-sidebar>

Method 2: JavaScript Creation

// Create sidebar with configuration
const sidebar = document.createElement('pushable-sidebar');
sidebar.setAttribute('position', 'left');
sidebar.setAttribute('width', '300px');
sidebar.setAttribute('min-width', '200px');
sidebar.setAttribute('max-width', '500px');
sidebar.setAttribute('theme', 'light');
sidebar.innerHTML = '<div>Sidebar content</div>';

// Add to document
document.body.appendChild(sidebar);

// Control programmatically
sidebar.toggle();       // Toggle expand/collapse
sidebar.setWidth('400px');
sidebar.setPosition('right');
sidebar.setTheme('dark');

Features

  • Push Content: The sidebar pushes the page content instead of overlapping it
  • Resizable: Drag the edge to resize the sidebar
  • Collapsible: Click the arrow to collapse/expand
  • Positioning: Place on left or right side
  • Theming: Light and dark themes included
  • Style Isolation: Uses Shadow DOM to prevent style conflicts
  • Responsive: Adapts to window resizing

API Reference

The library provides the following public methods:

  • toggle() - Toggles between collapsed and expanded states
  • collapse() - Collapses the sidebar
  • expand() - Expands the sidebar
  • setWidth(width) - Sets the width (e.g., '300px', '20em')
  • setPosition(position) - Sets position ('left' or 'right')
  • setTheme(theme) - Sets theme ('light' or 'dark')

Attributes you can set on the custom element:

  • position - 'left' or 'right' (default: 'left')
  • width - Width when expanded (default: '300px')
  • min-width - Minimum width when resizing (default: '50px')
  • max-width - Maximum width when resizing (default: '500px')
  • collapsed - Initial state (default: 'false')
  • collapsed-width - Width when collapsed (default: '30px')
  • z-index - z-index value (default: 9999)
  • theme - 'light' or 'dark' (default: 'light')
